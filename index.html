Phage Genome Mapper

Genome map created using D3.js and coding sequence files. There are currently two phage genomes available.

Select a phage file

Scroll to zoom in, hover to view gene properties.

This map was created to fulfill the requirements of Math W82: Data Visualization with D3 at Calvin University, January 2020. The CDS files were created and parsed with a python script.
